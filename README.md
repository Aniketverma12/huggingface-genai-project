# Hugging Face GenAI & NLP Project

A beginner-friendly Generative AI and NLP project built with **Python** and **Hugging Face Transformers**.

## Features
- Sentiment Analysis
- Text Generation
- Question Answering
- English to Hindi Translation
- Image Classification

## Tech Stack
- Python
- Transformers
- PyTorch
- Pillow
- Requests

## Project Structure
```bash
huggingface-genai-project/
│── README.md
│── requirements.txt
│── main.py
│── .gitignore
```

## Installation
```bash
pip install -r requirements.txt
```

## Run
```bash
python main.py
```

## Resume Description
Built NLP and Generative AI workflows using Hugging Face Transformers for sentiment analysis, text generation, question answering, translation, and image classification. Worked with pretrained Transformer models and performed inference for real-world GenAI tasks.
